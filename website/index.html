Fonctionnalités

Gestion des Streams

Interface web pour gérer tous vos streams avec suivi automatique de l'état live/offline des streamers via l'API Twitch.

  • Interface Vue.js
  • État en temps réel
  • API Twitch

Intégration OBS Studio

  • OBS WebSocket
  • Création automatique des sources
  • Chat Twitch

Récupération des données via l'API officielle du ZEvent avec suivi des dons, participants et statistiques en temps réel.

Installation

Démarrez en quelques minutes

1

Prérequis

Assurez-vous d'avoir Node.js, npm, OBS Studio avec WebSocket activé, et Git installés.

2

Cloner le projet

git clone https://github.com/fabienmillet/zevent-streamlink.git
3

Installation des dépendances

cd zevent-streamlink && npm install
4

Configuration

Copiez le fichier .env.example, configurez vos paramètres et activez WebSocket dans OBS Studio.

5

Démarrage

npm run dev

Accédez à http://localhost:5173

Démarrage rapide

Utilisez npm run dev pour lancer le backend et frontend simultanément.

Configuration OBS

N'oubliez pas d'activer WebSocket dans OBS : Outils → Paramètres du serveur WebSocket

URLs importantes

  • Interface : localhost:5173
  • API : localhost:3001
  • Docs : localhost:3001/docs
